# Step by Step

## Setup do Projeto

1. Usar o spring initializr
2. Criar as pastas
    - config
    - controllers
    - dto
    - entities
    - exceptions
    - interfaces
    - repositories
    - services
    - filters
3. Configurar o Cors
4. Configurar o Security
5. Criar o Dependencies Configuration
6. Criar a ResponseException
7. Criar o ExceptionHandlerController


## A cada exercício

1. Entity
2. DTO
3. Repository
4. Interface do Service
5. Service
6. Controller
